# 사용자 관리 (User Management)

## 개요

사용자 관리를 통해 관리자는 플랫폼 내의 사용자 액세스 및 역할을 제어할 수 있습니다. 사용자를 초대하고, 조직에 할당하고, 권한을 관리할 수 있습니다.

## 주요 기능

- **사용자 목록**: 모든 등록된 사용자와 그들의 역할 및 상태를 봅니다.
- **사용자 추가**: 새 사용자 계정을 생성하거나 멤버를 초대합니다.
- **역할 할당**: 시스템 관리자, 조직 관리자 또는 개발자와 같은 역할을 할당합니다.
- **사용자 상태**: 사용자 계정을 활성화하거나 비활성화합니다.
