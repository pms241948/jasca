# 스캔 결과 (Scan Results)

## 개요

프로젝트에 대해 수행된 보안 스캔의 상세 보고서입니다. 심각도, 상태 또는 스캐너 유형별로 결과를 필터링할 수 있습니다.

## 주요 기능

- **결과 필터링**: 치명적(Critical) 또는 높음(High) 문제에 집중합니다.
- **상세 보기**: 코드 스니펫 및 수정 제안을 검사합니다.
- **상태 업데이트**: 문제를 오탐(False Positive) 또는 수정됨(Fixed)으로 표시합니다.
