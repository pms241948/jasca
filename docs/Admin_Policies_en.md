# Policy Management

## Overview

Policy Management is crucial for defining and enforcing security standards across the organization. Administrators can create policies that dictate how vulnerabilities are handled and blocked.

## Key Features

- **Policy List**: View active and inactive security policies.
- **Create Policy**: Define new rules based on severity, CWE, or other criteria.
- **Enforcement**: Apply policies to specific projects or the entire organization.
