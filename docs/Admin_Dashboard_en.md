# Admin Dashboard

## Overview

The Admin Dashboard provides a centralized view of the system's status, including key metrics, recent activities, and quick access to essential management tools. It allows administrators to monitor the overall health and security posture of the organization.

## Key Features

- **System Overview**: View aggregated statistics such as total users, projects, and active policies.
- **Recent Activity**: Track the latest system events and user actions.
- **Quick Links**: Rapidly navigate to frequently used admin sections like User Management or Project Management.
- **Vulnerability Summary**: Get a high-level summary of critical vulnerabilities across the organization.
