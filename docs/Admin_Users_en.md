# User Management

## Overview

User Management enables administrators to control user access and roles within the platform. You can invite users, assign them to organizations, and manage their permissions.

## Key Features

- **User List**: View all registered users with their roles and status.
- **Add User**: Create new user accounts or invite members.
- **Role Assignment**: Assign roles such as System Admin, Org Admin, or Developer.
- **User Status**: Activate or deactivate user accounts.
