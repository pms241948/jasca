# 조직 관리 (Organization Management)

## 개요

조직 관리 섹션에서는 관리자가 Jainsight 내의 조직 구조를 관리할 수 있습니다. 여기에는 새 조직 생성, 세부 정보 업데이트 및 관련 설정 관리가 포함됩니다.

## 주요 기능

- **조직 목록**: 모든 조직의 페이징된 목록을 봅니다.
- **조직 생성**: 시스템에 새 조직을 추가합니다.
- **세부 정보 수정**: 조직 이름, 설명 및 기타 메타데이터를 업데이트합니다.
- **조직 삭제**: 필요한 경우 내부 조직을 제거합니다.
