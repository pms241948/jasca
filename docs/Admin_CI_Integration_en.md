# CI Integration

## Overview

Manage integrations with Continuous Integration (CI) systems like Jenkins, GitHub Actions, or GitLab CI. Generate webhooks and tokens for automated scanning in pipelines.

## Key Features

- **Integration Status**: View connected CI systems.
- **Webhook Management**: specific webhooks for receiving scan triggers.
- **Token Generation**: Create tokens for CI/CD pipelines to authenticate.
