# 관리자 대시보드 (Admin Dashboard)

## 개요

관리자 대시보드는 시스템 상태에 대한 중앙 집중시 뷰를 제공합니다. 주요 지표, 최근 활동 및 필수 관리 도구에 대한 빠른 액세스를 포함하여, 관리자가 조직의 전반적인 상태와 보안 태세를 모니터링할 수 있도록 합니다.

## 주요 기능

- **시스템 개요**: 총 사용자 수, 프로젝트 수, 활성 정책과 같은 집계된 통계를 확인합니다.
- **최근 활동**: 최신 시스템 이벤트 및 사용자 작업을 추적합니다.
- **빠른 링크**: 사용자 관리 또는 프로젝트 관리와 같이 자주 사용하는 관리 섹션으로 빠르게 이동합니다.
- **취약점 요약**: 조직 전체의 중요한 취약점에 대한 상위 수준 요약을 제공합니다.
